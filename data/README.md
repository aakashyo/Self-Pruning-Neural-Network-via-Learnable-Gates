# CIFAR-10 Dataset Directory

This directory serves as the automatic download location for the CIFAR-10 dataset binaries.

**Note:** You do not need to search for or download the dataset manually. When you run `solution.py`, PyTorch's `torchvision` library will automatically download, extract, and structure the CIFAR-10 dataset securely into this folder if it senses the data is missing.

This file acts as a structural placeholder dummy so the `data/` directory exists natively when the repository is cloned by reviewers.
